# Sales Prediction Project

## Overview
This project focuses on predicting sales using machine learning techniques.

## Dataset
A structured dataset with advertising features was used and preprocessed.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Process
- Data cleaning (handling missing values and duplicates)
- Model training using Linear Regression
- Prediction and visualization of results

## Output
The model compares actual vs predicted sales using a scatter plot.
